Trabalho Prático: Torre de Hanói
 Nesse repositório estão os arquivos referentes ao trabalho da disciplina "Desenvolvimento Web", da Universidade Federal de Juiz de Fora, ministrada pelo professor Igor Knop.

 O objetivo do trabalho é aplicar os conhecimentos obtidos na disciplina acerca, puramente, da linguagem de marcação HTML, da linguagem de estilização CSS e da linguagem de programação JavaScript (e, por consequência, a manipulação do DOM).

 O mesmo consiste na criação autoral de uma simulação do quebra-cabeça matemático "Torre de Hanói". O jogo consiste em uma base contendo três pinos e 8 discos de tamanhos diferentes que começam empilhados em ordem decrescente de tamanho no primeiro pino, formando uma pirâmide.

 O objetivo é transferir toda a torre de discos do pino de origem para um pino de destino, usando o pino do meio como auxiliar. No final, a torre deve ficar exatamente igual a como começou, mas em outro pino.

 As regras do jogo:
1. Apenas um disco pode ser movido por vez.

2. Um disco maior nunca pode ficar em cima de um menor.

3. Discos em repouso devem sempre estar em um dos pinos.

 Tudo isso — o ambiente do jogo, os pinos, os discos, os elementos visuais e dinâmicos, bem como o histórico de jogadas, informações cumulativas e regras — foi definido única e exclusivamente com os recursos fornecidos pela disciplina e pelo aprofundamento do estudo na mesma.
